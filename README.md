Symphonique Lagos
This is the codebase for the Symphonique Lagos website. It’s a dedicated space for a music academy based here in Lagos, built to be fast, responsive, and easy to navigate.

The site is built using Next.js and focuses on a clean, intentional design that reflects the academy’s creative energy.

## Tech Stack
Framework: Next.js (App Router)

Styling: Tailwind CSS

Language: TypeScript

Animations: Framer Motion

Icons: Lucide React

## Getting Started
If you’re setting this up locally, just follow these steps:

Clone the project

Bash
git clone https://github.com/your-username/symphonique-lagos.git
Install dependencies

Bash
npm install
Run the development server

Bash
npm run dev
Open http://localhost:3000 in your browser to see the site.

## Project Structure
src/app: Handles all the routing and page layouts.

src/components: The building blocks of the UI (buttons, cards, etc.).

src/styles: Where the global styles and Tailwind config live.

public: Stores all the images, fonts, and icons.

## Deployment
The site is set up to deploy on Vercel. Any changes pushed to the main branch will update the live site automatically.
